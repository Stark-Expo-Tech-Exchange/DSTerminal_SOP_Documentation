DSTerminal - Defensive Security Terminal v2.1.0
=================================================

A comprehensive security analysis and monitoring tool.

FEATURES:
- System vulnerability scanning
- Network monitoring with threat detection
- File encryption/decryption
- Forensic analysis tools
- Report generation

INSTALLATION:
1. Run the installer as Administrator
2. Choose installation options
3. Launch from Start Menu or desktop

FIRST STEPS:
1. Run 'help' to see available commands
2. Configure your workspace with 'workspace init'
3. Set up encryption with 'encrypt-setup'
