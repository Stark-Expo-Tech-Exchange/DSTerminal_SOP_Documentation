Quick Start Guide

DSTerminal is a comprehensive Security Operations Center (SOC) platform designed for defensive security operations. Type help in the terminal to see all available commands.

✓ System Ready - DSTerminal is fully operational with all security modules loaded.

First-Time Setup

# Initialize workspace
mkdir DSTerminal_Workspace
cd DSTerminal_Workspace

# Create default directories
mkdir operators scans reports exploits sandbox quarantine

# Start DSTerminal
python dsterminal.py

Basic Command Reference

system scan -All Core Security
Performs comprehensive system security scan including memory analysis, process inspection, file integrity checks, and network monitoring.
system scan -All
--quick Fast scan --deep Thorough analysis --output Save report
net -n mon Network
Real-time network monitoring with threat detection. Analyzes connections, bandwidth usage, and identifies suspicious activities.
net -n mon
--duration Monitor time --export Save report --geoip Enable geolocation

Command Categories

Category Command Description
System sysinfo Display detailed system information
Network portsweep <target> Scan for open ports
Forensics memdump Capture memory for analysis
Crypto encrypt <file> AES-256 encryption
Hardening harden -t sys Apply system hardening

Installation Guide

Windows Installation

# Using the installer
1. Download DSTerminal_Installer_v2.1.0.exe
2. Run as normal user (no admin required)
3. Follow the setup wizard
4. Launch from Start Menu or Desktop

Linux Installation

# From source
git clone https://github.com/Stark-Expo-Tech-Exchange/DSTerminal_releases_latest.git
cd DSTerminal_releases_latest
pip install -r requirements.txt
python dsterminal.py

macOS Installation

# Using Homebrew
brew install dsterminal
dsterminal
